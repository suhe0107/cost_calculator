# 开店盈利性测算工具

> 🚀 **一句话推荐**：本项目全程由 **Trae IDE** 辅助开发，是一款专为实体店老板打造的“开店盈利性测算”神器，支持理想与现实双维度对比，助您算清每一笔账，科学决策，理性创业！

## 📖 项目介绍

在实体创业的浪潮中，很多老板往往因为账目算不清而陷入困境。本项目**“开店盈利性测算”**是一个轻量级的单页 Web 应用（HTML/CSS/JS），旨在帮助创业者全方位、多维度地评估开店成本与预期收益。

无需安装任何软件，下载即可在浏览器中运行，简单、高效、透明。

## ✨ 核心功能

*   **📊 双视角对比**：同时展示“理想数值”与“现实数值”，让风险与机遇一目了然，避免盲目乐观。
*   **💰 全链路成本核算**：
    *   **收入预估**：客单价、日单量、税率及毛成本率。
    *   **运营成本**：房租（精确到单价/面积）、人工（人数/日薪）、水电物业等。
    *   **投资摊销**：支持转让费、装修、设备等前期投入按年限自动摊销到每一天。
*   **📈 智能利润分析**：自动计算保本线（盈亏平衡点）、日/月/年纯利以及预计回本周期。
*   **📄 便捷导出**：支持一键生成高清 PDF 报表，方便打印、存档或与合伙人分享。

## 🛠️ 使用方法

1.  克隆或下载本项目到本地。
2.  直接双击打开 `cost_calculator.html` 文件（推荐使用 Chrome 或 Edge 浏览器）。
3.  根据实际情况输入各项数据，系统将实时计算并显示结果。
